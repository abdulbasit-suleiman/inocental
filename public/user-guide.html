üöÄ Getting Started

1. Accessing the Application

Open your web browser and navigate to the application URL. You'll see the main dashboard with two options:

  • Voter Data Extractor: For creating and managing voter registration sheets
  • Admin Dashboard: For administrators to view all data (requires special access)

2. Voter Data Extractor

Click on the "Voter Data Extractor" card to begin creating voter registration sheets.

üìã Step-by-Step Guide

  1. Create a New Excel Sheet

    Click on "Voter Data Extractor" from the main dashboard. Enter a name for your Excel sheet (e.g., "Ward 5 Voter Registrations") and click "Create Excel Template".

    The app will automatically check if a sheet with this name already exists for you. If it exists, it will load the existing sheet. If not, it will create a new sheet.

  2. Capture Voter Forms

    Position the voter registration form in front of your camera. Click the camera icon to capture the image. Review the image to ensure all text is clearly visible.

  3. Extract Data

    Click "Extract Data" to process the image using OCR. The system will automatically recognize and extract voter information including surname, middle name, first name, phone number, gender, ward, unit, and NIN.

  4. Review and Edit

    Check the extracted data in the form fields. Make any necessary corrections to ensure accuracy, then click "Add to Excel Sheet" to save the entry.

  5. Continue or Save

    Repeat the process for additional voter forms. Click "Download Sheet" to get your Excel file locally or "Save to Firebase" to store your data in the cloud.

üí° Tips for Best Results

Image Quality

Ensure good lighting when capturing forms. Keep the form flat and aligned with the camera. Make sure all text is clearly visible and not blurry. Avoid shadows or glare on the form.

Data Verification

Always review extracted data before adding to your sheet. Pay special attention to phone numbers (should be 10 digits), names (check for OCR misreads), and NIN numbers (verify format).

üîß Troubleshooting

Poor OCR Results

Solution: Recapture the image with better lighting and alignment. Ensure the text is horizontal and not skewed.

Upload Failures

Solution: Check your internet connection and try again. If the problem persists, refresh the page and retry the upload.

Missing Fields

Solution: Manually enter missing information in the form fields. The system will still save your complete entry.

Firebase Permission Errors

Solution: These are now handled gracefully by the application. The app will continue working by creating new sheets even if it cannot check for existing ones due to permissions.

CORS Errors When Saving to Firebase

Solution: Firebase Storage requires CORS configuration for localhost development. See the Firebase Rules Setup documentation for instructions on configuring CORS.

Technical Requirements

  • Modern web browser (Chrome, Firefox, Safari, Edge)
  • Camera access (for image capture)
  • Stable internet connection (for cloud storage)
  • Properly configured Firebase rules and CORS (for cloud storage features)
